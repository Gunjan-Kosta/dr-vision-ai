**Dr. Vision AI** is an advanced, specialized medical dashboard built to analyze retinal images for Diabetic Retinopathy. By utilizing deep learning prediction engines alongside advanced visual feedback tools like Grad-CAM heatmaps, it assists Ophthalmologists in diagnosing patients quicker and more accurately. 

🌐 **Live Demo:** [https://Gunjan-Kosta.github.io/dr-vision-ai/](https://Gunjan-Kosta.github.io/dr-vision-ai/)

---

## 🚀 Key Features

* **Advanced Retinal Analysis**: Upload clinical ocular scans to instantly generate a Diabetic Retinopathy Severity reading (None, Mild, Moderate, Severe, Proliferative).
* **Grad-CAM Visualization**: Employs heatmaps to visually isolate anomalous anomalies such as microaneurysms and hemorrhages, validating the AI's predictions visually.
* **Clinical Report Generation (PDF / Print)**: Intelligently formats highly-detailed medical diagnosis reports optimized explicitly for physical printouts and PDFs, omitting unneeded web logic.
* **Prescription Engine**: Seamlessly converts diagnosis severities, custom precautions, and medication timelines into hospital-ready prescriptions.
* **Dark Mode & Deep Glassmorphism**: Provides a premium, deep-space themed clinical UI. Specifically designed to lower optical fatigue during prolonged analysis sessions.

## 🛠️ Technology Stack

* **Frontend Framework**: React 19 / Vite
* **Styling**: Tailwind CSS & Vanilla CSS specific overrides (Autofill normalization, Print layout adjustments)
* **Animation Engine**: Framer Motion
* **Iconography**: Lucide React
* **Hosting / CI Pipeline**: GitHub Pages 

## 💻 Local Development

If you'd like to run or test the UI/UX flows locally on your machine, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Gunjan-Kosta/dr-vision-ai.git
   ```

2. **Navigate to the directory and install dependencies**
   ```bash
   cd dr-vision-ai
   npm install
   ```

3. **Start the local development server**
   ```bash
   npm run dev
   ```
   *Your app will launch locally on `http://localhost:5173`.*
